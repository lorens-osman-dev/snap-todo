## ⚡ 1. The Quick Capture Philosophy
**1.** Hit `Alt + T`  
**2.** Write  
**3.** Press `Enter`  


## ⚡ 2. The Fluidity System

Snap-todo features a completely mouse-free architecture. Every UI interaction is bound to a hardened, Wayland-safe Clutter key-focus pipeline. We call this the **Fluidity System**.

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| **Toggle Snap-todo** | `Alt + T` *(Customizable)* | Instantly opens the dropdown menu or slide-out drawer. Focus is automatically trapped in the text entry. |
| **Navigate** | `Up` / `Down` / `Tab` | Fluidly move focus through your active and completed tasks without ever reaching for the mouse. |
| **Reorder Tasks** | `Alt + Up/Down` | Dynamically drag and drop rows up or down your list. Enforces strict boundaries to prevent crossing pinned/completed states. |
| **Toggle Status** | `Space` | Mark a task as completed or active. Focus intelligently recalculates and hands off to the next visual successor. |
| **Delete Task** | `Delete` | Instantly remove a task and seamlessly hand off keyboard focus to the adjacent row. |
| **Dismiss UI** | `Esc` | Instantly close the menu or drawer and return to your workspace. |





## ⚡ 3. Dual UI Modes
Choose the layout that fits your workflow.

1. Normal Menu: A traditional, lightweight dropdown menu attached directly to your top bar.

2. Slide-out Drawer: A modern, full-height Wayland-native side panel that slides in for maximum screen real estate.
